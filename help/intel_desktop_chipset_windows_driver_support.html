1.  Introduction

This Web page lists the Intel 3/4/5/6/7/8/9/100 Series Chipset (Desktop) and the Intel X58 Express/X79 Express/X99 Chipset driver support for Windows XP/Vista/7/8/8.1/10, including the Intel desktop chipset's chipset, LAN, audio, and graphics devices.

In the table below:

  • Discontinued = Intel is no longer developing and publicly releasing updated drivers for the Intel desktop chipset.
  • Active = Intel is still developing and publicly releasing updated drivers for the Intel desktop chipset.

2.  Intel 3/4/5/6/7/8/9/100 Series Chipset (Desktop) And Intel X58 Express/X79 Express/X99 Chipset Driver Support For Windows XP/Vista/7/8/8.1/10

In the table below:

  • + = Intel publicly released a driver for the device. Therefore, before installing Windows, the user can go to the motherboard manufacturer web site and download the latest driver for the device. Then, after installing Windows, the user can install the previously downloaded driver.
  • - = Intel did not publicly release a driver for the device. Therefore, the user cannot go to the motherboard manufacturer web site and download a driver for the device. Instead, the user is reliant on Windows and/or Windows/Microsoft Update providing a driver for the device.

    If the version of Windows is newer than the Intel desktop chipset, then Windows and/or Windows/Microsoft Update typically provides a satisfactory driver for the Intel desktop chipset's chipset, LAN, audio, and graphics devices. Concerning graphics, if the display's initial screen resolution is less than the display's native screen resolution, place the computer online and allow Windows to automatically install a driver specific for the graphics device, which, in the case of Windows 8/8.1/10 typically occurs within 30 minutes of placing the computer online and is apparently independent of Windows/Microsoft Update. Upon the installation of the device specific graphics driver, the display's screen resolution automatically changes to the display's native screen resolution, and the device listed under Device Manager | Display adapters changes from the generic plug-n-play Microsoft Basic Display Adapter to a device specific entry such as Intel G33/G31 Express Chipset Family (Microsoft Corporation - WDDM 1.0).

    If the Intel desktop chipset is newer than the version of Windows, then Windows can only provide a generic plug-n-play driver, if any driver, for the Intel desktop chipset's chipset, LAN, audio, and graphics devices. Concerning chipset and graphics, Windows typically provides a generic plug-n-play driver that allows the device to operate, but at a sub-optimal performance rate. For better performance, run Windows/Microsoft Update and hope that Intel provided Microsoft with a device specific driver for the device to download and install via Windows/Microsoft Update. Concerning LAN and audio, Windows typically cannot provide a driver for the device and, therefore, the device does not operate, which, unfortunately, in the case of LAN, precludes running Windows/Microsoft Update to download and install any available device drivers.

(1) (2) (3)  Starting with the Intel 9 Series Chipset (Desktop), Intel stopped manufacturing desktop motherboards based on their own desktop chipsets. This means there are no Intel motherboards based on the Intel 9 Series Chipset (Desktop), Intel X99 Chipset, and Intel 100 Series Chipset (Desktop). In addition, although Intel still produces the drivers, and the driver updates, for these chipsets, Intel also stopped providing the drivers for these chipsets for download from Intel's downloadcenter.intel.com. As a result, the only way to obtain the drivers for these chipsets is from the third party motherboard manufacturer web site. Motherboard manufacturers are not required to manufacture motherboards that support all of the editions (e.g., 7 or 10) and versions (i.e., 32-bit or 64-bit) of Windows that the Intel desktop chipset drivers support. For example, say Intel provides Chipset XYZ drivers for Windows 7 32-bit and 64-bit, Windows 8 32-bit and 64-bit, Windows 8.1 32-bit and 64-bit, and Windows 10 32-bit and 64-bit. However, to reduce cost, a motherboard manufacturer might decide to manufacture motherboards based on Chipset XYZ only for Windows 7 32-bit and 64-bit, Windows 8.1 64-bit, and Windows 10 64-bit, not for Windows 8 32-bit and 64-bit, Windows 8.1 32-bit, nor Windows 10 32-bit. With Intel dropping out of the motherboard manufacturing business, Windows 8 being replaced by Windows 8.1, and Windows 8.1 being replaced by Windows 10, this is true now more than ever. As a result, the Intel 9 Series Chipset (Desktop), Intel X99 Chipset, and Intel 100 Series Chipset (Desktop) driver support for Windows XP/Vista/7/8/8.1/10 listed above is theoretical based on; 1.) the versions of Windows within Microsoft Extended Support at the time the chipsets were released (except for Vista, which even the Intel 8 Series Chipset (Desktop) does not support), and 2.) the future versions/editions of Windows likely to be supported by the release of updated Intel desktop chipset drivers. Therefore, before purchasing a motherboard, check the motherboard manufacturer web site and confirm the availability of Intel desktop chipset driver downloads for your edition and version of Windows.
