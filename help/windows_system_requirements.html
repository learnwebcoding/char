1.  Introduction

Windows system requirements define the minimum and sometimes recommended hardware required to install and run Windows. The primary Windows system requirements are for processor, memory, and hard disk. This page lists the Windows 9x/ME/2000/XP/Vista/7/8/8.1/10 system requirements for processor, memory, and hard disk.

1.1.  Abbreviations

  • 9x = Windows 95 and Windows 98 Second Edition.
  • 98 = Windows 98 = Windows 98 Second Edition.
  • ME = Windows ME = Windows Millennium Edition.
  • 2000 = 2000 Pro = Windows 2000 Pro = Windows 2000 Professional.
  • Vista HB = Windows Vista Home Basic.
  • Vista HP, Bus, Ent, Ult = Windows Vista Home Premium, Business, Enterprise, Ultimate, respectively.
