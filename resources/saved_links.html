1.1.  Adobe Flash Player

i
Adobe Flash Player for:
  • Internet Explorer on Windows XP/Vista/7: 1.) is not integrated into Internet Explorer, including Internet Explorer 10 for Windows 7; 2.) can be uninstalled via Control Panel | Add or Remove Programs (Windows XP) or Control Panel | Programs: Uninstall a program (Windows Vista/7); and 3.) updates, by default, are automatically installed without user interaction by the Adobe Flash Player Updater.
  • Firefox, SeaMonkey, and Opera on Windows XP/Vista/7: 1.) is not integrated into Firefox, SeaMonkey, or Opera; 2.) can be uninstalled via Control Panel | Add or Remove Programs (Windows XP) or Control Panel | Programs: Uninstall a program (Windows Vista/7); and 3.) updates, by default, are automatically installed without user interaction by the Adobe Flash Player Updater.
  • Internet Explorer on Windows 8: 1.) is integrated into Internet Explorer; 2.) cannot be uninstalled via Control Panel | Programs: Uninstall a program; and 3.) updates, by default, are automatically installed without user interaction by Windows Update.
  • Chrome 5.0+ on Windows XP/Vista/7/8: 1.) is integrated into Chrome; 2.) cannot be uninstalled via Control Panel | Add or Remove Programs (Windows XP) or Control Panel | Programs: Uninstall a program (Windows Vista/7/8); and 3.) updates are automatically installed without user interaction by the Google Installer (GoogleUpdate.exe).

13.7.  Windows XP: Recovery Console

i
The Windows XP Recovery Console:
  • is a command-line environment with restricted file/folder access from which the user issues commands to repair a Windows XP installation.
  • can be started by booting the computer from the Windows XP product CD.
  • can be installed from the Windows XP product CD and then started from the Windows XP boot menu that appears during Windows XP Startup.
  • is replaced by Windows Vista/7 System Recovery Options.
